# Personal Portfolio

A responsive and modern portfolio built using React, Vite, and Tailwind CSS. It showcases my projects, experience, skills, and provides easy access to external profiles and contact links.

---

## Live Website
https://portfolio-e7gt.onrender.com/

---

## Tech Stack

- Frontend: React 19, Vite, Tailwind CSS
- Routing: React Router v7
- UI & Icons: Lucide, React Icons
- Animation: Custom transitions, shimmer loaders, aurora visuals  
- Hosting: Render

---

## Features

- Single-page application with smooth navigation
- Clean project and experience presentation
- Responsive and mobile-friendly layout
- Lazy-loaded routes for performance optimization
- Resume download and professional profile links
- Optional contact integration using Formspree
